Este es un proyecto de práctica para aprender Git y GitHub de Programación 1.



**Nombre**: José Osorio



**Descripcion del proyecto**

Este proyecto lo hice como práctica para aprender Git y GitHub. Aprendí a clonar repositorios, hacer commits y subir archivos. También tuve algunos errores como problemas con el push y la configuración del usuario, pero al final logré solucionarlo y subir todo correctamente.

**Lo que aprendi**

Aprendí cómo funciona Git y GitHub para guardar proyectos, subir cambios y llevar control de versiones. También entendí cómo hacer commits y subir archivos al repositorio, además de solucionar errores comunes como problemas con el push y la configuración del usuario.



Cambio hecho en la rama jose.
